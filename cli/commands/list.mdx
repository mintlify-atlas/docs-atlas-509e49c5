## Overview

The `list` command displays the top 50 most popular fonts available on NoBadFonts, sorted by download count. Use this to discover fonts before adding them to your project.

## Syntax

```bash
npx nobadfonts list
```

<Note>
No arguments are required for the `list` command.
</Note>

## Example

```bash
npx nobadfonts list
```

## Font Categories

Fonts are tagged with descriptive categories:

### Style Categories
- **sans-serif** - Clean, modern fonts without serifs
- **serif** - Traditional fonts with decorative strokes
- **display** - Decorative fonts for headlines
- **monospace** - Fixed-width fonts for code
- **handwriting** - Script and handwritten styles

### Character Categories
- **modern** - Contemporary, clean design
- **geometric** - Based on geometric shapes
- **humanist** - Organic, calligraphic influence
- **elegant** - Sophisticated and refined
- **friendly** - Approachable and warm
- **technical** - Industrial and precise

## What It Does

<Steps>
  <Step title="Queries the API">
    Fetches font data from the NoBadFonts database via Supabase REST API.
  </Step>
  
  <Step title="Sorts by Popularity">
    Orders fonts by download count (most popular first).
  </Step>
  
  <Step title="Limits Results">
    Returns the top 50 fonts to keep output manageable.
  </Step>
  
  <Step title="Displays Metadata">
    Shows font names with their tags and categories for easy browsing.
  </Step>
</Steps>

## API Request Details

The command makes a GET request to:

```
{SUPABASE_URL}/rest/v1/fonts?select=name,category,tags&limit=50&order=downloads.desc
```

**Headers:**
- `apikey`: Your Supabase anonymous key
- `Authorization`: Bearer token with anonymous key

## Using the Output

Once you've found a font you like, copy its exact name and use it with other commands:

### Add via CSS Import

```bash
npx nobadfonts add "Inter"
```

### Download Locally

```bash
npx nobadfonts download "Inter"
```

<Note>
Font names are case-insensitive, so "Inter", "inter", and "INTER" all work.
</Note>

## Error Handling

### API Connection Error

If the CLI can't reach the NoBadFonts API:

```bash
⠋ Fetching available fonts...
✖ Error fetching fonts: Failed to fetch
```

**Common causes:**
- Missing or invalid `SUPABASE_URL`
- Missing or invalid `SUPABASE_ANON_KEY`
- Network connectivity issues
- API is temporarily unavailable

See [Configuration](/cli/configuration) and [Troubleshooting](/cli/troubleshooting) for solutions.

### Empty Response

If the API returns no fonts:

```bash
⠋ Fetching available fonts...

Available Fonts (Top 50):

To install (CSS Import): npx nobadfonts-cli add "Font Name"
To download (Local Files): npx nobadfonts-cli download "Font Name"
```

This usually indicates a database issue or incorrect API configuration.

## Filtering and Search

The `list` command shows all top 50 fonts. To find specific fonts:

### Use grep (Unix/Linux/macOS)

```bash
npx nobadfonts list | grep -i "sans"
```

Shows only sans-serif fonts.

### Use findstr (Windows)

```bash
npx nodfonts list | findstr /i "mono"
```

Shows only monospace fonts.

### Search by Category

```bash
npx nobadfonts list | grep -i "serif"
```

Filters fonts containing "serif" in their metadata.
