# Quick Start

Get up and running with NoBadFonts in just a few minutes. This guide covers the essential steps to start using curated fonts in your projects.

## Browse fonts on the web

The fastest way to explore NoBadFonts is through the web interface at [nobadfonts.in](https://nobadfonts.in).

<Steps>
  <Step title="Open the font catalog">
    Navigate to the [Font Catalog](https://nobadfonts.in) to browse the curated collection.
  </Step>

  <Step title="Filter and search">
    Use the filter panel to narrow by category:
    - **Sans Serif** - Clean, modern typefaces
    - **Serif** - Traditional, editorial fonts
    - **Display** - Bold, attention-grabbing styles
    - **Monospace** - Fixed-width fonts for code

    Sort options: Trending, Popular, Editor Picks, Newest, Alphabetical
  </Step>

  <Step title="Preview in context">
    Click any font to see context-aware previews:
    - **UI Context** - Dashboard interfaces with buttons and tooltips
    - **Editorial Context** - Multi-column text for readability
    - **Code Context** - Syntax-highlighted blocks

    Enter custom text to preview with your own content.
  </Step>

  <Step title="Download or share">
    - Click **Download** to get font files
    - Use **Share** to create a shareable link
    - Click the heart icon to favorite fonts
  </Step>
</Steps>

## Install fonts with the CLI

For developers, the CLI provides instant font installation from your terminal.

### Installation

<CodeGroup>

```bash npm
npm install -g nobadfonts-cli
```

```bash npx
npx nobadfonts-cli
```

```bash pnpm
pnpm add -g nobadfonts-cli
```

</CodeGroup>

### Install your first font

There are two ways to add fonts to your project:

<Tabs>
  <Tab title="CSS Import (Recommended)">
    The `add` command generates a lightweight CSS import:

    ```bash
    npx nobadfonts-cli add "Inter" "Satoshi"
    ```

    This creates a `fonts.css` file that references fonts from the NoBadFonts CDN:

    ```css fonts.css
    @font-face {
      font-family: 'Inter';
      src: url('https://obpmvhzprzfovybfgcxs.supabase.co/storage/v1/object/public/font-files/...');
      font-weight: 400;
      font-style: normal;
    }
    /* Additional variants... */
    ```

    Import it in your project:

    ```html
    <link rel="stylesheet" href="fonts.css">
    ```

    Or in your CSS:

    ```css
    @import url('./fonts.css');
    ```
  </Tab>

  <Tab title="Local Files">
    The `download` command saves font files locally:

    ```bash
    npx nobadfonts-cli download "Inter" "Satoshi"
    ```

    Files are saved to `./fonts/` with this structure:

    ```
    fonts/
    ├── inter/
    │   ├── Inter-Regular.woff2
    │   ├── Inter-Bold.woff2
    │   └── Inter-Italic.woff2
    ├── satoshi/
    │   ├── Satoshi-Regular.woff2
    │   └── Satoshi-Bold.woff2
    └── fonts.css
    ```

    Use the generated CSS file to reference local fonts.
  </Tab>
</Tabs>

### List available fonts

See all curated fonts:

```bash
npx nobadfonts-cli list
```

Output:

```
Available Fonts (Top 50):
• Inter (sans, geometric, modern)
• Satoshi (sans, display, bold)
• Space Grotesk (sans, monospace, technical)
• Cabinet Grotesk (sans, display, elegant)
...
```

## Use fonts in your project

Once fonts are installed, use them in your CSS:

<Tabs>
  <Tab title="CSS">
    ```css
    body {
      font-family: 'Inter', sans-serif;
    }

    h1 {
      font-family: 'Satoshi', sans-serif;
      font-weight: 700;
    }
    ```
  </Tab>

  <Tab title="Tailwind CSS">
    Add fonts to your `tailwind.config.js`:

    ```js tailwind.config.js
    module.exports = {
      theme: {
        extend: {
          fontFamily: {
            sans: ['Inter', 'sans-serif'],
            display: ['Satoshi', 'sans-serif'],
          },
        },
      },
    }
    ```

    Use in your markup:

    ```html
    <h1 class="font-display text-4xl font-bold">
      Beautiful Typography
    </h1>
    ```
  </Tab>

  <Tab title="React / Next.js">
    Import the CSS file in your app:

    ```tsx app.tsx
    import './fonts.css';

    export default function App() {
      return (
        <div style={{ fontFamily: 'Inter, sans-serif' }}>
          <h1 style={{ fontFamily: 'Satoshi, sans-serif' }}>
            Beautiful Typography
          </h1>
        </div>
      );
    }
    ```
  </Tab>
</Tabs>

## Available commands

| Command | Description |
|---------|-------------|
| `add <fonts...>` | Generate CSS import for CDN-hosted fonts |
| `download <fonts...>` | Download font files locally |
| `list` | Display all available fonts |
