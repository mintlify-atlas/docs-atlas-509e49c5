# Curated Typography for Modern Interfaces

NoBadFonts is a next-generation font discovery platform that eliminates the noise from typography selection. We believe finding the right typeface shouldn't involve wading through thousands of low-quality options.

## Why NoBadFonts?

Traditional font repositories are cluttered with poorly spaced, unfinished, and non-production-ready typefaces. **NoBadFonts** focuses exclusively on:

- **Usable fonts** - Production-ready typefaces with complete character sets
- **Clean typography** - Professionally designed with proper spacing and kerning
- **Real design workflows** - Tools built for how designers actually work

## Core Features

### Context-Aware Font Tester

Preview fonts in the exact environments they'll be used - not just "The quick brown fox". Test typefaces in:

- **UI Context** - Simulated dashboards with tooltips, buttons, and data tables
- **Editorial Context** - Dense, multi-column text for readability testing
- **Code Context** - Syntax-highlighted blocks for monospace fonts

### Smart Upload Pipeline

Our WASM-powered upload system automatically:
- Validates font files and metadata
- Converts TTF/OTF to optimized WOFF2 in-browser
- Manages variants (Regular, Bold, Italic, etc.)
- Generates unique slugs to prevent collisions

### Font Pairing Tool

Create beautiful typography combinations with live preview. Pair heading and body fonts with real-time rendering across multiple contexts.

### CLI for Developers

Install fonts instantly from your terminal:

```bash
npx nobadfonts add inter
```

Works on macOS, Windows, and Linux. No manual downloads required.

## Technology Stack

Built with modern web technologies:

- **React 19** - Cutting-edge UI framework
- **Vite** - Lightning-fast build tool
- **Tailwind CSS v4** - Rapid styling with sub-millisecond compile times
- **Supabase** - Serverless PostgreSQL with Row Level Security
- **GSAP & Framer Motion** - Buttery smooth animations
- **Capacitor** - Native mobile apps from a shared codebase
