## Get All Fonts

Retrieve a list of fonts from the collection.

```bash
GET /rest/v1/fonts
```

### Query Parameters

<ParamField query="select" type="string" default="*">
  Specify which columns to return. Use `*` for all columns, or comma-separated column names.
  
  Example: `select=id,name,designer,category`
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Maximum number of rows to return. Range: 1-1000
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of rows to skip (for pagination)
</ParamField>

<ParamField query="order" type="string">
  Sort results by column. Use `.desc` suffix for descending order.
  
  Example: `order=downloads.desc` or `order=name.asc`
</ParamField>

### Filtering Parameters

<ParamField query="name" type="string">
  Filter by font name. Supports operators:
  - `eq.value` - exact match
  - `ilike.%value%` - case-insensitive pattern match
  
  Example: `name=ilike.%inter%`
</ParamField>

<ParamField query="category" type="string">
  Filter by font category.
  
  Example: `category=eq.sans-serif`
</ParamField>

<ParamField query="designer" type="string">
  Filter by designer name.
  
  Example: `designer=ilike.%rasmus%`
</ParamField>

<ParamField query="is_published" type="boolean" default="true">
  Filter by publication status.
  
  Example: `is_published=eq.true`
</ParamField>

<ParamField query="is_featured" type="boolean">
  Filter for featured fonts only.
  
  Example: `is_featured=eq.true`
</ParamField>

<ParamField query="is_trending" type="boolean">
  Filter for trending fonts only.
  
  Example: `is_trending=eq.true`
</ParamField>

<ParamField query="tags" type="string">
  Filter by tags array. Use `cs.{value}` for contains.
  
  Example: `tags=cs.{modern,geometric}`
</ParamField>

### Request Example

```bash
curl "https://YOUR_PROJECT_ID.supabase.co/rest/v1/fonts?select=*&limit=10&order=downloads.desc" \
  -H "apikey: YOUR_ANON_KEY" \
  -H "Authorization: Bearer YOUR_ANON_KEY"
```

## Get Single Font

Retrieve a specific font by ID or slug.

### By ID

```bash
GET /rest/v1/fonts?id=eq.{font_id}&select=*
```

### By Slug

```bash
GET /rest/v1/fonts?slug=eq.{font_slug}&select=*
```

### Request Example

```bash
curl "https://YOUR_PROJECT_ID.supabase.co/rest/v1/fonts?slug=eq.inter-display&select=*" \
  -H "apikey: YOUR_ANON_KEY" \
  -H "Authorization: Bearer YOUR_ANON_KEY"
```

## Get Font with Variants

Retrieve a font with all its variants using embedded resources.

```bash
GET /rest/v1/fonts?id=eq.{font_id}&select=*,font_variants(*)
```

```bash
curl "https://YOUR_PROJECT_ID.supabase.co/rest/v1/fonts?slug=eq.inter-display&select=*,font_variants(*)" \
```

## Search Fonts

Use the `search_fonts` RPC function for advanced full-text search.

```bash
POST /rest/v1/rpc/search_fonts
```

### Request Body

<ParamField body="query" type="string">
  Search query string
</ParamField>

<ParamField body="filter_tags" type="array">
  Array of tags to filter by
</ParamField>

<ParamField body="sort_by" type="string">
  Sort option: `trending`, `popular`, `newest`, `alphabetical`, `editor-picks`, `featured`
</ParamField>

<ParamField body="limit_val" type="integer" default="16">
  Number of results to return
</ParamField>

<ParamField body="offset_val" type="integer" default="0">
  Number of results to skip
</ParamField>

### Request Example

```bash
curl "https://YOUR_PROJECT_ID.supabase.co/rest/v1/rpc/search_fonts" \
  -X POST \
  -H "apikey: YOUR_ANON_KEY" \
  -H "Authorization: Bearer YOUR_ANON_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "modern",
    "filter_tags": ["geometric", "sans-serif"],
    "sort_by": "trending",
    "limit_val": 10,
    "offset_val": 0
  }'
```

### Response Example

```json
[
  {
    "font_data": {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "name": "Inter Display",
      "designer": "Rasmus Andersson",
      "category": "sans-serif",
      "tags": ["modern", "geometric", "ui"]
    }
  }
]
```

## Usage Example (JavaScript)

Here's how the NoBadFonts application queries fonts:

```typescript
import { supabase } from '../lib/supabase'
import type { Font } from '../types/font'

// Search fonts with filters
const { data, error } = await supabase.rpc('search_fonts', {
  query: 'modern',
  sort_by: 'trending',
  filter_tags: ['geometric', 'sans-serif'],
  limit_val: 16,
  offset_val: 0
})

// Get font with variants
const { data: fontData } = await supabase
  .from('fonts')
  .select('*, font_variants(*)')
  .eq('slug', 'inter-display')
  .order('order_index', { foreignTable: 'font_variants', ascending: true })
  .single()
```

<Note>
  The code above is from `src/hooks/useFonts.ts:35` and `src/hooks/useFont.ts:24`
</Note>
